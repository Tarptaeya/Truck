# Truck Language Guide

### Declaring variable
```javascript
var x = 1
var y = x * x
var z = true and false
var w = "hello"
```

### Conditionals
**If Statement**
```javascript
if 4 > 2 {
    print("4 is greater than 2")
} else {
    print("2 is greater than 4")
}
```

**While Statement**
```javascript
var i = 0
while i <= 10 {
    print(i)
    i = i + 1
}
```

**Break and Continue**
```javascript
var i = 0
while true {
    if i >= 10 {
        break
    }
    i = i + 1
}
```

### Functions
**Function declaration**
```javascript
var sum = fn (x, y) {
    return x + y
}
```

### Built In Functions
**Print**
```javascript
print("Hello World")
```

**Input**
```javascript
var x = input()
```

**Type**
```javascript
var x = "message"
type(x)
```

**Type converison**
```javascript
var x = num("24")
var y = "message " + num(32)
```
