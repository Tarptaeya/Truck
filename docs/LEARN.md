# Truck Language Guide

### Declaring variable
```javascript
var x = 1
var y = 20
var z = x * y + 31
var w = true
```

### Conditionals
**If Statement**
```javascript
if 4 > 2 {
    print 4
} else {
    print 2
}
```

**While Statement**
```javascript
var i = 0
while i <= 10 {
    print i
    i = i + 1
}
```

**Break and Continue**
```javascript
var i = 0
while true {
    if i >= 10 {
        break
    }
    i = i + 1
}
```

### Functions
**Function declaration**
```javascript
var sum = fn (x, y) {
    return x + y
}
```

### Utilities
**Print Statement**
```javascript
var x = 10
print x
```
